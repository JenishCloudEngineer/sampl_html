Welcome

This example demonstrates inline styling using the style attribute on HTML elements. Inline styles are useful for quick tweaks or generated HTML, but for larger projects prefer internal or external stylesheets.

  • Responsive layout with a tiny media query (in <style> block).
  • Cards, buttons, and nav links all styled inline.
  • Hover uses a CSS class because inline styles cannot express :hover.

Quick tips

  1. Use inline styles for generated HTML or emails.
  2. Move repeated styles to a stylesheet to avoid duplication.
  3. Remember: pseudo-classes require CSS rules, not inline style attributes.
